## 📂 Dataset

This project uses the *Fake and Real News Dataset* from Kaggle.

The dataset consists of two main files:

* **Fake.csv**: contains news articles labeled as fake
* **True.csv**: contains news articles labeled as real

Each record includes multiple features such as:

* Title of the article
* Full text content
* Subject/category
* Publication date

The dataset contains thousands of real-world news articles, making it suitable for training and evaluating machine learning models for text classification tasks.

Due to its relatively large size, the dataset is not included in this repository.

You can download it from Kaggle:
https://www.kaggle.com/datasets/mdkalimullahzakir/fake-and-real-news-dataset

After downloading, place the following files in the project directory:

* Fake.csv
* True.csv
